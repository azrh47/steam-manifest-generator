Nerai Templates - Steam Manifest Generator
==========================================

Generated by Nerai Templates (https://nerai.xyz/)
Educational template for Steam manifest generation

Files Included:
--------------
- appmanifest_{appid}.acf     - App manifest template
- depot_1_template.manifest   - Depot manifest template (Windows)
- depot_2_template.manifest   - Depot manifest template (Mac/Linux)
- {appid}_script_template.lua - Lua script template
- key.vdf                     - Steam key VDF template

Template Information:
--------------------
All templates are educational examples that use real Steam API data.
They are designed to help understand Steam manifest structure and are
NOT functional Steam files.

Lua Script Usage:
---------------
The Lua script uses standard Steam functions:
- addappid(appid) - Adds app ID to manifest
- setManifestid(depotid, manifestid) - Sets manifest ID for depot

Key VDF Format:
---------------
The key.vdf file contains educational Steam key templates in VDF format.
Keys are generated in XXXXX-XXXXX-XXXXX format for educational purposes.
